Project Overview
This project performs Exploratory Data Analysis on the Iris Dataset using Python.
The goal is to understand the dataset by inspecting its structure and visualizing feature relationships and distributions.
The analysis is implemented in a Jupyter-style notebook using Google Colab, making it easy to run and reproduce.

Objectives
The project demonstrates how to:
1) Load a dataset using Pandas
2) Inspect dataset structure and statistics
3) Visualize relationships between features
4) Understand feature distributions
5) Identify potential outliers

📊 Dataset Description
   The Iris Dataset is a well-known dataset used for learning data science and machine learning.
   It contains measurements of iris flowers from three species:
   1) Setosa
   2) Versicolor
   3) Virginica
      
Dataset Features
1) Feature	Description
2) sepal_length	Sepal length in cm
3) sepal_width	Sepal width in cm
4) petal_length	Petal length in cm
5) petal_width	Petal width in cm
6) species	Flower species
   Total samples: 150

🛠 Tools & Libraries
1) The analysis uses the following Python libraries:
2) Pandas – Data loading and analysis
3) Matplotlib – Plotting graphs
4) Seaborn – Advanced statistical visualization
5) The notebook was developed and executed in Google Colab.

📈 Analysis Steps

1️⃣ Data Loading
The dataset is loaded using Pandas or directly through Seaborn.

2️⃣ Data Inspection
Basic dataset exploration includes:
1) Checking dataset shape
2) Viewing column names
3) Displaying the first few rows
4) Understanding data types
5) Generating statistical summaries
   
3️⃣ Data Visualization
Several visualizations are used:
1) Scatter Plot
2) Shows relationships between numerical features and species categories.
3) Histogram
4) Displays the distribution of numerical variables.
5) Box Plot
6) Helps identify outliers and understand data spread.

Key Learnings
This project demonstrates fundamental data analysis skills, including:
1) Dataset inspection
2) Statistical summaries
3) Feature visualization
4) Outlier detection
These techniques are commonly used in machine learning workflows before model training.
